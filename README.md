# 📦 Order Management API

API REST desenvolvida em Java com Spring Boot durante o curso do Nélio Alves, com o objetivo de praticar a construção de aplicações web utilizando Spring, JPA/Hibernate e banco de dados relacional.

## 📚 Conceitos aplicados

- Spring Boot
- Spring Data JPA
- Hibernate
- API REST
- Arquitetura em camadas
- Injeção de Dependência
- CRUD
- Modelagem de banco de dados
- Relacionamentos entre entidades
- Tratamento de exceções

## ⚙️ Funcionalidades

- Gerenciamento de usuários
- Gerenciamento de pedidos
- Gerenciamento de produtos
- Gerenciamento de categorias
- Associação entre pedidos e produtos
- Registro de pagamentos
- Operações CRUD via API REST

## 🛠️ Tecnologias

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- H2 Database
- Git
- GitHub

## 🎯 Objetivo

Este projeto foi desenvolvido para consolidar os conhecimentos em desenvolvimento de APIs REST utilizando Spring Boot, aplicando conceitos de arquitetura em camadas, persistência de dados com JPA/Hibernate e modelagem de banco de dados relacional.
